# Dotfiles

These are my dotfiles

# Dependencies

- bash: shell
- bspwm: window manager
- dmenu: running apps
- dunst: notifications
- fd & fzf: fuzzy searching
- hsetroot: wallpaper
- lemonbar: bar
- picom: compositor
- sxhkd: key daemon

# Optional dependencies
- newsboat: for rss
- pass: for password management
- st: as the secondary terminal application
